## 2.
三角形の3辺の長さを入力すると、「直角三角形です。」又は「直角三角形ではありません。」と出力するプログラムを作成せよ．<br><br>
out
```
3辺の長さを入力してください。
```
in
```
3 4 5
```
out
```
直角三角形です。
```

ans
```cpp
#include <iostream>
using namespace std;

int main(){
    int Sides[3];
    cout << "3辺の長さを入力してください。" << endl;
    for(int i=0;i<3;i++){
        cin >> Sides[i];
        Sides[i]*=Sides[i];
    }

    if(
        sides[0] == sides[1] + sides[2] ||
        sides[1] == sides[0] + sides[2] ||
        sides[2] == sides[0] + sides[1]
    )       cout << "直角三角形です。";
    else    cout << "直角三角形ではありません。";
    cout << endl;

    return 0;
}
```
